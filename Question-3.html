Grid auto row

The grid-auto-rows property sets a size for the rows in a grid container. This property affects only rows with the size not set.

Grid auto column

The grid-auto-columns property in CSS grid is used to specify the size of columns that are automatically created by the grid when you place elements that are not explicitly placed in specific columns. In other words, it defines the size of any "implicit" columns that are created by the grid.
